Dropdown

¿Que es un Dropdown?

Un Dropdown es una lista desplegable que se puede mostrar o ocultar en el sitio web.

Estructura básica de un Dropdown

Pon el puntero del ratón arriba de este texto

Hello World!

<div class="dropdown">
  <span>Pon el puntero del ratón arriba de este texto</span>
  <div class="dropdown-content">
    <p>Hello World!</p>
  </div>
</div>
